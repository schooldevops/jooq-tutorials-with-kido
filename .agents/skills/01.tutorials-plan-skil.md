# 챕터별 강의 계획 수립

이 스킬은 각 챕터와 아이템 마다 강의 작성 계획을 제시합니다. 제시된 강의는 실행 절차에 따라 강의를 생성하고, 검토후 체크를 수행한다. 

## 실행 절차
1. **컬리큘럼 제목 분석**: 컬리큘럼 제목을 분석합니다.
2. **강의 계획 작성**: 학습목표, 각 학습 목표를 달성하기 위한 학습 내용, 학슴 내용을 실제 실행해볼 수 있도록 하기 위한 기술 선택, 개발환경 구성, 실제 개발 및 테스트 단계등을 계획한다.
3. **강의 생성**: 계획된 강의를 생성한다.
4. **강의 검토**: 생성된 강의를 검토한다.
5. **강의 체크**: 검토된 강의를 체크한다.

## 주의 사항
- 각 단계마다 산출믈을 작성하고, 체크 후 승인을 받고 다음을 진행할것
- 강의 계획은 각 챕터 디렉토리를 생성하고, md 파일을 작성한다.
- 동작 방식을 설명할 때에는 반드시 mermaid 를 활용하여 흐름을 작성하고, 강의 내용을 설명할때 mermaid 를 활용하여 시퀀스 다이어그램이나 BPMN 다이어그램을 작성할것
- 소스 코드는 02.develop-code-skil.md를 활용하여 생성할것
