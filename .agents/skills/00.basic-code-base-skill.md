# 개발 방향

이 스크리은 java-spring과 kotlin-spring 2개의 기본 코드 베이스를 생성하는 스킬이다.

## 기술 스택

- java-spring
    - java
    - spring boot
    - gradle
    - jooq
    - postgresql
    - lombok
    - jackson
- kotlin-spring
    - kotlin
    - spring boot
    - gradle
    - jooq
    - postgresql
    - lombok
    - jackson

## 주의 사항
- 코드베이스를 작성은 반드시 정상적으로 수행되는 코드 베이스여야 한다. 
- 기술 스택을 정확히 지켜져야한다.
