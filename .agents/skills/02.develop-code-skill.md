# 개발 스킬

이 스킬은 강의 내용을 실제 프로그램으로 개발하는 데 집중합니다.
개발은 다음 절차에 따라 작성한다. 

## 실행 절차
1. **강의내용 파악**: 각 챕터와 하위 아이템에 따른 강의 내용을 파악한다. 
2. **강의 내용을 실제 실행하기 위해서 기본 코드베이스를 복사**: 기본 코드베이스는 개발을 위한 java-spring 기반의 코드 베이스와, koktlin-spring 기반의 코드 베이스 가지며, 매번 개발때마다 기본 코드 베이스를 복사하여 복사한 곳에서 강의 내용을 추가 적성한다.
3. **개발 단계마다 개발 플랜 작성**: 모든 코드를 생성할때에는 분할 정복 방식으로 개발을 해야한다. 이러한 분할정복 방식을 낭려하고, 스케줄에 따라서 개발이 되도록 플랜을 작성한다.
3. **BDD 방식의 코드작성**: 플랜 내에는 반드시 TDD 형식이나 BDD 형식으로 테스트를 기반으로 코드가 개발이 되어야하며, 코드 커버리지는 항상 지켜져야하고, 이전 테스트 코드가 깨지지 않도록 작성해야한다.

## 주의 사항
- 플랜은 매 단계마다 테스트 코드작성, 실제코드 작성, 테스트 완료후 다음 진행 여부를 확인해야한다.
- 매 챕터마다 작성해야하는 코드는 기본 코드베이스를 반드시 복사하여 개발이 되어야하며, 개발 코드는 반드시 테스트가 되어야한다.
